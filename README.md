🎲 Digital Dice using 4017 Counter

📌 About
This is a Digital Electronics mini project that simulates a dice using a 4017 decade counter and 7-segment display.

⚙️ Components
IC 4017
555 Timer
7 Segment Display
Resistors & Capacitors

🔌 Working
Clock pulses from the 555 timer drive the 4017 counter.
The outputs are mapped to display numbers from 1 to 6.
